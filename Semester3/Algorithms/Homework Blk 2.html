import numpy as np
import matplotlib.pyplot as plt

def LagrangeFormInterpolation(knots, values, x):
    """
    Compute the Lagrange form interpolation at points x given the knots and values.
    """
    n = len(knots)
    P = np.zeros_like(x, dtype=float)
    for i in range(n):
        L = np.ones_like(x)
        for j in range(n):
            if j != i:
                L *= (x - knots[j]) / (knots[i] - knots[j])
        P += values[i] * L
    return P

# Define the function
def f(x):
    return np.log(x)

# Define the interval and the degree of the polynomial
a, b = 1, 2
n = 9

# Generate 10 equally spaced nodes in the interval [1, 2]
x_nodes = np.linspace(a, b, n + 1)
y_nodes = f(x_nodes)

# Generate 400 equally spaced points in the interval [1, 2] for evaluation
x_eval = np.linspace(a, b, 400)
f_eval = f(x_eval)

# Compute the Lagrange interpolation polynomial at the evaluation points
p9_eval = LagrangeFormInterpolation(x_nodes, y_nodes, x_eval)

# Compute the maximum absolute error
max_error = np.max(np.abs(f_eval - p9_eval))

print("Maximum Absolute Error:", max_error)

# Plot the original function and the interpolation polynomial

The second interpolation error theorem states that the error $ E_n(x) $ of the function $f(x)$ with the interpolation polynomial $ P_n(x) $ is given by: $$ \left| f(x) - P_n(x) \right| \leq \frac{1}{4(n+1)} \left| f^{n+1}(\xi) \right| h^{n+1} \leq \frac{1}{4(n+1)} M h^{n+1} $$ where:

  • $ M $ is the upper bound of the $(n+1)$-th derivative of $ f(x) $ on the interval $[a, b]$.
  • $ h = \frac{b - a}{n} $ is the spacing between the nodes.
  • $ n $ is the degree of the interpolation polynomial.

for the function: $$ f(x) = \ln(x) $$ first we Calculate the upper bound $ M $ of the $(n+1)$-th derivative of $ f(x) $:

Given that the absolute value of the $ n $-th derivative of $ f(x) $ is: $$ |f^{(n)}(x)| = \frac{(n-1)!}{x^n} $$

For $ n = 9 $, we need to find the 10th derivative of $ f(x) $: $$ |f^{(10)}(x)| = \frac{9!}{x^{10}} $$

The maximum value of $ |f^{(10)}(x)| $ in the interval $[1, 2]$ is: $$ \max_{x \in [1, 2]} |f^{(10)}(x)| = \frac{9!}{1^{10}} = 9! $$

$\Rightarrow M = 9! $.

And Interval of interpolation $h$ : $$ h = \frac{2 - 1}{9} = \frac{1}{9} $$ $$ n = 9 $$ The interpolation error is: $$ |f(x) - P_9(x)| \leq \frac{1}{4(9+1)} M h^{9+1} $$ $$ |f(x) - P_9(x)| \leq \frac{1}{4 \cdot 10} \cdot 9! \cdot \left(\frac{1}{9}\right)^{10} $$ $$ |f(x) - P_9(x)| \leq \frac{1}{4 \cdot 10 } \cdot 9! \cdot \left(\frac{1}{9}\right)^{10} $$ $$ |f(x) - P_9(x)| \leq \frac{1}{40} \cdot 9!\left(\frac{1}{9}\right)^{10} $$ $$ |f(x) - P_9(x)| \leq \frac{1}{40} \cdot 9! \frac{1}{9^{10}} $$ $$ |f(x) - P_9(x)| \leq \frac{1}{40} \cdot 9!\frac{1}{3486784401} $$ $$ |f(x) - P_9(x)| \leq \frac{1}{139471376040} $$

$$= 2.6018241900469027594459517601817e-6$$

Hence: The theoretical upper bound of the interpolation error for $ p_9 $ is approximately $ 2.602 \times 10^{-12} $.

Deriving the $ n $-dependent upper bound of the absolute error for the composite Simpson's rule, we use the error term for Simpson's rule. The error term for the composite Simpson's rule is given by:

$$ E(f) = -\frac{(b - a)h^4}{180} f^{(4)}(\xi) $$

where $ h = \frac{b - a}{n} $ and $ \xi $ is some point in the interval $[a, b]$.

For the integral $ \int_0^\pi \sin(x) \, dx $, the fourth derivative of $ \sin(x) $ is $ \sin(x) $, and its maximum absolute value on the interval $[0, \pi]$ is 1. Therefore, the error term simplifies to:

$$ E(f) \leq \frac{\pi h^4}{180} $$

Since $ h = \frac{\pi}{n} $:

$$ E(f) \leq \frac{\pi \left(\frac{\pi}{n}\right)^4}{180} $$ $$ E(f) \leq \frac{\pi^5}{180 n^4} $$ This gives us the $ n $-dependent upper bound of the absolute error for the composite Simpson's rule.

import numpy as np
import matplotlib.pyplot as plt

# Function to perform Simpson's Rule
def MySimpson(f, a, b, n):
    # Check if n is even, as Simpson's Rule requires it
    if n % 2 != 0:
        print("Error: Number of subintervals (n) must be even.")
        return np.nan
    else:
        print(f"Number of subintervals (n) is even: {n}")
    
    h = (b - a) / n  # Width of each subinterval
    integral = f(a) + f(b)  # Start with the endpoints
    
    # Sum the terms with coefficient 4 (for odd indexed points)
    for i in range(1, n, 2):
        integral += 4 * f(a + i * h)
    
    # Sum the terms with coefficient 2 (for even indexed points)
    for i in range(2, n, 2):
        integral += 2 * f(a + i * h)
    
    # Multiply by h/3
    integral *= h / 3
    
    return integral

# Define the integrand function
def integrand(x):
    return np.sin(x)
